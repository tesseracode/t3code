## Description

Expose desktop-managed local and WSL environments as a first-class Managed environments mode in the Connections Add Environment dialog, dependent on the Windows WSL integration. The feature should restore detected WSL/local managed environment candidates without hiding them under Pairing URL, while keeping main-process IPC surface area minimal through a small adapter/interface.
