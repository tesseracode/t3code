# Feature Request: Add support for system-generated turns: allow external events (file watchers, CI callbacks, MCP notifications, webhooks) to automatically trigger a new model turn in a provider session without explicit user input. This would extend the orchestration layer to accept synthetic turn requests alongside user-initiated ones, enabling autonomous agent workflows, background monitoring, and event-driven AI responses.

**Slug**: `system-generated-turns`
**Created**: 2026-04-28T07:32:15Z

## Description

Add support for system-generated turns: allow external events (file watchers, CI callbacks, MCP notifications, webhooks) to automatically trigger a new model turn in a provider session without explicit user input. This would extend the orchestration layer to accept synthetic turn requests alongside user-initiated ones, enabling autonomous agent workflows, background monitoring, and event-driven AI responses.
